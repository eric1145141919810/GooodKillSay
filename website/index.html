为什么选择 Goood KillSay？

🔄

刷屏模式

每15秒自动发送一条消息，随机选择在线玩家替换占位符

⚔️

击杀模式

仅在击杀时发送消息，通过正则表达式智能识别击杀信息

🔀

混合模式

同时开启刷屏和击杀模式，让你随时都能成为焦点

⌨️

自定义按键

在游戏设置中自由配置按键，按K开关、M切换模式、R重载

💾

配置保存

自动保存你的开关状态和模式选择，重启游戏无需重新设置

🔧

灵活配置

通过配置文件支持多种服务器的击杀格式，完全自定义

如何安装 Goood KillSay

1

准备工作

确保已安装 Minecraft 1.21.4、Fabric Loader 和 Fabric API

2

下载文件

下载 killsay-1.0.0.jar、killsay.txt 和 killpatterns.json

3

安装模组

将 jar 文件放入 .minecraft/mods 文件夹

4

放置配置

将 killsay.txt 和 killpatterns.json 放入游戏根目录

5

开始游戏

启动游戏，按 K 键开启，享受你的 PVP 时光！

配置说明

📄

killsay.txt

存放你的消息内容，每行一条。使用 {name} 作为玩家名称的占位符。 

GG {name}!
Nice try {name}!
Better luck next time {name}!
Ez {name}
L {name}
📋

killpatterns.json

配置击杀消息的识别规则，支持自定义正则表达式。

[
  {
    "name": "sw1",
    "pattern": "^(.*) 被 (.*) 击败$",
    "killerGroup": 2,
    "victimGroup": 1
  }
]
